Inštalácia WordPress

🚀 Rýchla inštalácia (XAMPP):

  1. Stiahnite XAMPP: apachefriends.org
  2. Spustite: Apache + MySQL servery
  3. Stiahnite WordPress: wordpress.org/download
  4. Vytvorte databázu: http://localhost/phpmyadmin → nová databáza "socinovacie"
  5. Nainštalujte: http://localhost/socinovacie
  6. Nahrajte tému: wp-content/themes/socialne-inovacie/
Alternatívy:
  • Local by Flywheel - Jednoklikové WordPress prostredie
  • MAMP - Mac/Windows lokálny server
  • WordPress.com - Online hosting
